# ChatRoom

Salle de chat en temps réel.

Connexion, envoi et réception messages avec date/heure/avatar utilisateur, possiblité de changer d'avatar.

Front: React, Redux, Axios, Semantic UI.

Back: Node.js, Express.js, Socket.io.

![](chatroom.png)

## Lancement

Front:
```
yarn
yarn start
```

Back:
```
yarn
yarn start
```
